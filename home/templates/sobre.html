Sobre

Projeto

O projeto foi meu primeiro projeto desenvolvido usando as tecnologias python com o framework django para back-end e html, css, javascript com os frameworks jQuery e Bootstrap 4 para front-end. Todo o código do projeto está disponível no meu github.

Votos

A página "Deixe seu Voto" foi criada apenas para ter uma noção dos candidatos de interesse das pessoas que acessaram o site e se houve alguma mudança após conhecerem melhor os candidatos. Nenhuma informação é armazenada, além das preenchidas pelo usuário. Notem também que essas informações não tem valor para comparação com as pesquisas reais, visto que ela só leva em conta as poucas pessoas que chegaram a ter acesso a esse site.

Os gráficos da página de votos foram gerados usando as bibliotecas d3, dc e crossfilter do javascript e são atualizados automaticamente com a submissão do formulário. O código também está disponível no github em home/templates/vote.html O código responsável por atualizar a tabela está presente no github de Sudeep Mandal.

Perfis de Candidato

As nuvens de palavras presentes foram geradas a partir dos planos de governo de cada candidato. Primeiro convertendo o pdf para arquivo de texto usando uma alteração do script pdfparser.py e em seguida usando o arquivo de texto para gerar a nuvem com o script generate_wordcloud.py também estará presente no meu github.
